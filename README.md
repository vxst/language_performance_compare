# Simple Performance compare for 3 languages

Performance benchmark for a simple computing and pointer intensive program in Java, C# and C++.

# Results

 * Java: 11.64s
 * C#:   8.38s
 * C++:  6.66s
 * C++ with reference-counting weak gc: 8.27s
